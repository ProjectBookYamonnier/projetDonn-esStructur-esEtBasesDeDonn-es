DTD & XML

Code DTD

Cette DTD définit une structure pour une bibliothèque de médias (comme des films ou des séries). La bibliothèque contient un titre, un éventuel glossaire, et plusieurs éléments MEDIA qui représentent les médias eux-mêmes. Chaque MEDIA contient un titre, des informations sur le nombre de saisons, le nombre d'épisodes, la durée moyenne des épisodes, l'âge recommandé, l'année de sortie, la durée totale, la catégorie, les acteurs impliqués, les sujets abordés (PARLE_DE), et un éventuel glossaire.

La DTD définit également les éléments pour le glossaire, les mots et leurs définitions, ainsi que les éléments pour les acteurs et les personnages joués (JOUE_DANS). Les éléments ACTEUR, nom et prénom sont utilisés pour représenter les informations relatives aux acteurs. Les éléments CATEGORIE et PARLE_DE sont des éléments de texte simples qui représentent respectivement la catégorie et les sujets abordés dans un média.

La DTD utilise des attributs pour stocker des informations supplémentaires sur certains éléments, comme l'auteur d'un glossaire et le type de média.

<?xml version="1.0" encoding="UTF-8"?>
<!ELEMENT BIBLIOTHEQUE (titre, glossaire*, MEDIA+)>
<!ELEMENT glossaire (mot+)>
<!ATTLIST glossaire auteur CDATA #IMPLIED>
<!ELEMENT mot (#PCDATA)>
<!ATTLIST mot def CDATA #REQUIRED>
<!ELEMENT MEDIA (titre, nb_saisons?, nb_ep?, duree_moy_ep?, categorie_age, annee_sortie, duree?, CATEGORIE, JOUE_DANS, PARLE_DE*, glossaire*)>
<!ATTLIST MEDIA type CDATA #REQUIRED>
<!ELEMENT titre (#PCDATA)>
<!ELEMENT nb_saisons (#PCDATA)>
<!ELEMENT nb_ep (#PCDATA)>
<!ELEMENT duree_moy_ep (#PCDATA)>
<!ELEMENT categorie_age (#PCDATA)>
<!ELEMENT annee_sortie (#PCDATA)>
<!ELEMENT duree (#PCDATA)>
<!ELEMENT ACTEUR (nom, prenom)>
<!ELEMENT nom (#PCDATA)>
<!ELEMENT prenom (#PCDATA)>
<!ELEMENT JOUE_DANS (ACTEUR+)>
<!ELEMENT CATEGORIE (#PCDATA)>
<!ELEMENT PARLE_DE (#PCDATA)>

Code XML

Ce fichier XML représente une bibliothèque de médias contenant des films et des séries, avec des informations telles que les titres, les acteurs, les catégories, les années de sortie et les durées. Le fichier commence par des déclarations XML et DTD, suivies des éléments principaux de la bibliothèque, chacun étant un élément de type "MEDIA". Les médias sont classés en fonction de leur type (Film ou Série). Chaque MEDIA contient des informations spécifiques telles que le titre, le nombre de saisons/épisodes pour les séries, la durée moyenne des épisodes, la catégorie d'âge et une liste d'acteurs. Il y a également des glossaires pour expliquer certains termes clés tels que les genres de films et de séries.
